For finite set S, $P = (s_1,\ldots s_j)$ with $s_i \in S$ is an ordered set partition fo S if
1) $S_i \cap S_k = \emptyset \quad \forall i\neq k$

2) $\overset{j}{\underset{i=1}{\cup}} S_i = S$




If a set S is partitioned into [[Disjoint Union | disjoint]] subsets S_1, S_2, \dots, S_k
	$S = S_1 \sqcup S_2 \sqcup \dots \sqcup S_k$
Then counting |S| reduces to summing the sizes:
	$|S| = |S_1| + |S_2| + \dots + |S_k|$
